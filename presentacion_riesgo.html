Objetivos del Proyecto

¿Qué resolvimos?

Análisis Cuantitativo

  • Cálculo en tiempo real de 6 indicadores técnicos.
  • Modelado de volatilidad condicional (ARCH/GARCH).
  • Pruebas empíricas de normalidad (Jarque-Bera).

Gestión de Riesgo

  • Cálculo de VaR y CVaR (Histórico, Paramétrico, Monte Carlo).
  • Optimización de Portafolios (Frontera de Markowitz).
  • Integración Macroeconómica (FRED).
Ingeniería de Software

Arquitectura Desacoplada

Backend (FastAPI)

  • Lógica aislada: Cálculos matemáticos y consumo de APIs.
  • Pydantic: Validación estricta (ej. Pesos suman 100%).
  • Rendimiento: Caché y endpoints asíncronos.

Frontend (Streamlit)

  • Interactividad: Dashboard dinámico.
  • Visualización: Gráficos interactivos con Plotly.
  • UX: Navegación por módulos (Pestañas).
Integración de Datos

Fuentes y Procesamiento

Yahoo Finance API (yfinance)

Extracción dinámica de series de tiempo financieras para un portafolio de activos tecnológicos (AAPL, MSFT, GOOGL, AMZN, TSLA).

FRED API (Federal Reserve)

Consumo de datos macroeconómicos oficiales para nutrir el modelo CAPM (Tasa Libre de Riesgo) y contextualizar el tracking error del portafolio.

Resultados

Proyecto 100% Funcional

Se cumplió a cabalidad con la rúbrica exigida, incorporando modelos financieros avanzados en un ecosistema de desarrollo moderno (Python 3.12). Las bonificaciones de arquitectura REST y señales macroeconómicas garantizan un producto de nivel profesional.
